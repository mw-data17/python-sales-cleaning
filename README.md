# Python Sales Cleaning

Mini-projekt pokazujący podstawowe przetwarzanie danych w Pythonie (pandas).
Skrypt wczytuje dane sprzedażowe z CSV, wykonuje proste czyszczenie danych oraz
oblicza prowizję i kwotę netto.

# Zakres
- wczytanie danych z CSV
- czyszczenie danych (braki, spacje, rekordy zwrócone)
- obliczenie prowizji
- zapis oczyszczonych danych do nowego CSV
- prosty raport w konsoli

# Technologie
- Python
- pandas
